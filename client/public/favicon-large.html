About This Favicon

The Guest Code favicon is designed as a stylized "G" with QR code elements, representing the app's focus on event access and digital tickets. The design uses SVG format for crisp rendering at any size.

The favicon is implemented with multiple sizes to ensure compatibility across all devices and platforms:

  • SVG vector format for modern browsers
  • PNG fallbacks at 192×192, 512×512 for PWA support
  • 16×16, 32×32 for traditional browser tabs and bookmarks
  • Apple Touch Icon for iOS devices

The color scheme matches the app's branding with #ffc807 (gold) on a black background.
